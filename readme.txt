Для запуска проекта необходимо установить на компьютер: node.js, git (для удобства)
1.Git
Проект также лежит по этой ссылке: https://github.com/NeverMind-V/fauna.git
Для скачивания необходимо в консоли написать git clone *имя ссылки*
2.Node.js
В консоли в папке проекта прописать:
    npm install --save-dev
    bower install --save-dev
3. Запуск
После загрузки и установки всех пакетов данных в консоли прописать команду gulp. Сайт будет развернут на локалхосте по адресу http://localhost:3000/
Для доступа к остальным страницам необходимо в адресной строке прописать название файла страницы
Главная - http://localhost:3000/
Контакты - http://localhost:3000/contacts.html
Аптеки - http://localhost:3000/pharmacy.html
Список товаров - http://localhost:3000/goods-list.html
Карточка товара - http://localhost:3000/goods.html
Список новостей - http://localhost:3000/article-list.html
карточка новости - http://localhost:3000/article.html
